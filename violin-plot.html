Violin Plot

Este tipo de grfico nos presenta una distribucin de probabilidad de una variable pero lo hace en vertical y con una representacin en espejo

A veces tambin puede superponerse una visualizacin de caja y bigotes sobre la de violn.

Es til cuando queremos comparar la distribucin de una variable sobre distintos grupos para comparar

import pandas as pd
import seaborn as sns

# Importamos el archivo csv

# Grfico de violn
plt.figure(figsize=(8, 6))

plt.title("Distribucin de Salarios por Experiencia")
